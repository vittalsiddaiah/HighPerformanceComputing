# Standalone version of matrix multiplication 3x3
## To Build the Code
```shell
# Copy the Sources (main.cpp, Matrix.hpp, StdHdr.hpp, StdLib.hpp, gcc.rc)
# Execute the script

./gcc.rc

#or

module load gcc/7.2.0
g++ ./main.cpp -O3 -std=c++17 -o ./mmply_gcc
module unload gcc/7.2.0

```
## The above commands should have generated the binary *mmply_gcc*
## The binary can be execute by:
```shell
./mmply_gcc
```
